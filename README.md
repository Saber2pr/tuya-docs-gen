# @saber2pr/tuya-docs-gen

> 自动生成组件库文档

```bash
npm i -g @saber2pr/tuya-docs-gen
```

### Usage

```bash
# tuya-docs-gen <demo-path> <docs-path>
tuya-docs-gen ./example/tuya-panel-kit ./dist
```
